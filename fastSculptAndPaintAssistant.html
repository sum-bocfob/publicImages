Fast Sculpt and Paint Assistant

このアドオンを使用すると、スカルプトとペイントのさまざまな操作を、 1 つのキーから簡単に呼び出せます。

キーボード上での面倒な指の移動を無くします。

これは、ペンタブレットユーザーにとって特に効果的です。
ペンタブのショートカット ボタンの数は限られていますが、左右の手をできるだけペンタブレットから離さずにスカルプト、ペイントできます。

メッシュスカルプト、テクスチャペイント、頂点ペイント、ウェイトペイント、カーブススカルプト、およびメッシュスカルプトメニューから作成されたオブジェクト用のメニューがあります。

バージョン1.8の新機能！頭身定規を作成！

使用方法

メッシュスカルプト

スカルプトモードでは、「Shift + U」 キー (設定から変更できます)を押すとメニューが開きます。

何もせずに閉じるには、同じキーを押すか、空白領域をクリックします。


メニューはいくつかのセクションに分かれています。


1: ブラシ半径 / 2: ブラシの強さ / 3:スムーズブラシの強さ

4: ビューパイを開く / 5: ブラシパネルを開く / 6: テクスチャパネルを開く / 7: ストロークパネルを開く / 8: 減衰パネルを開く / 9: ブラシコンテキスト(右クリックで開くメニュー)を開く / 10: 面セットパイメニューを開く / 11: ピボット設定パイメニューを開く / 12: 自動マスクパイメニューを開く / 13: リメッシュパネルを開く / 14: 対称パネルを開く

15: マスクを反転 / 16: マスクを縮小,拡大 / 17: マスクをシャープに,スムーズに / 18: マスクをクリア

19: ファイルを保存 / 20: ミラーモディファイアを追加 / 21: 対象化 / 22: 領域をズーム / 23: “前面のみ”のオンオフ / 24: 減衰の形状の切り替え

25: Z軸をロック / 26: Y軸をロック / 27: X軸をロック

28: 表示を反転 / 29: 全てのトポロジーを表示 / 30:トポロジーの短径範囲を隠す / 31: トポロジーのマスク範囲を隠す

32: 初期設定サイズでリメッシュ(プリファレンスから変更可能) / 33: 現在の設定サイズでリメッシュ / 34: ディテールサイズを取得してリメッシュ / 35: 現在のサイズ × mulltiplier2(プリファレンスから変更可能)の値でリメッシュ / 36: 現在のサイズ * multiplier1(プリファレンスから変更可能)の値でリメッシュ / 37: 現在のサイズ ÷ mulltiplier1の値でリメッシュ / 38: 現在のサイズ ÷ multiplier2の値でリメッシュ / 39: スマートマルチレゾリューション

40: 新規オブジェクトを作成 / 41: オブジェクトを結合 / 42: バックアップを作成 / 43: スカルプトモードを転送 / 44: オブジェクトモードに移行 / 45: アルファを消去への切り替え

46: 次のmatcap / 47: 前のmatcap / 48: 設定されたマットキャップの切り替え / 49: アクティブオブジェクトのワイヤーフレームを表示・非表示

50: 透過表示 / 51: シルエット表示 / 52: Matcapを反転 / 53: スムーズ,フラットシェーディングの切り替え / 54: アクティブオブジェクトの表示・非表示 / 55: コレクションの表示・非表示 / 56: 背景画像の透明度変更

57: 定規の作成 / 58: 全定規の表示 / 59: 全定規の非表示 / 60: 全定規の削除


また、スカルプトのボックスマスク, 投げ縄で隠す, クロスフィルター, 移動などのようなツールや、グリースペンシルのボックス, トリム, スポイトなどのツールも、ツールセットへ追加することができます。

まず追加したいツールをアクティブにし、ツールシェルフ上で右クリックし
”アクティブなツールをアクティブなツールセット/新しいツールセットへ追加”を押します。



アセットブラウザーからは、それぞれのモードのツールセットに対して、ブラシを追加できます。
また、複数のブラシをまとめて追加することもできます。

ツールセット、ブラシを選択

ツールセットにブラシを追加したら、3DビューポートでYキー(デフォルト)を押して、ブラシセレクターを開くことができます。
アイコンの上にマウスを持っていくと、ブラシ名が表示され、クリックするとそのブラシをアクティブにします。



複数のツールセットが存在する場合、ブラシセレクターを開いた状態でTまたはUキー(デフォルト)を押して、アクティブなツールセットを変更できます。
また、ブラシセレクター上部の四角形をクリックしても変更可能です。



開いたときと同じキー(デフォルトではY)を押すか、マウスを遠くに持っていくとブラシセレクターが閉じます。

ブラシは、上から時計回りに配置されます。
パネルから順序を変更したり、削除したりすることが可能です。

インポート / エクスポート

インポート / エクスポートメニューには、3種類のメニューが存在します。

グローバルライブラリは、異なるBlenderファイル間で、すべてのツールセットの保存 / 読み込みをします。
これは一つだけ作成できます。

「グローバルライブラリを読み込む」をクリックすると、現在のライブラリが上書きされることに注意してください。

グローバルライブラリは、それぞれのモード毎に保存されます。

ツールセットは、アクティブなツールセットをテキストファイルにエクスポートしたり、エクスポートしたテキストファイルからツールセットをインポートします。

ライブラリは、すべてのツールセットをテキストファイルにインポート / エクスポートします。

「ライブラリをインポート」をクリックすると、現在のライブラリが上書きされることに注意してください。

設定

設定にはいくつかの調整可能なパラメータがあります。


“シンプルパネルを使用”をオンにすると、ブラシセレクターは名前のみを表示するシンプルなパネルになります。
これはアクティブなツールセットのブラシのみを表示します(ショートカットキーでツールセットを変更することはできません)。
ブラシは左上から下に配置されます。



ブラシセレクターのアイコンは自動で生成されますが、何らかの理由でファイルが削除された場合などは、”アイコンの画像を再生成”ボタンで再び生成することができます。



Toolset Settingsでは、ツールセットの名前と順序を変更できます。



プリファレンス → アドオン → Quick Brush Selectorから、ショートカットキーなどを変更できます。

プリファレンスの「グローバルライブラリを自動で読み込む」をオンにしていれば(デフォルトでオン)、新しい.blendファイルでも自動で読み込まれます。

インストール方法

Blender4.2以上
  • zipファイルをダウンロード(解凍する必要はありません)
  • 編集→プリファレンス→”アドオン”または”エクステンションを入手”タブ
  • 右上の下向き矢印から、ディスクからインストールをクリック
  • ダウンロードしたzipファイルを選択
Blender4.1以下
  • zipファイルをダウンロード(解凍する必要はありません)
  • 編集→プリファレンス→アドオン
  • インストールをクリックし、zipファイルを選択
  • チェックボックスにチェックを入れる
