Fast Sculpt and Paint Assistant

This addon makes using various sculpt and paint operation easy in one key.

Remove tedious moving finger process across keyboard.

This is especially effective for pen tablet users.
While the number of shortcut buttons on the pen tab is limited, you will be able to sculpt and paint without taking your left and right hands off the pen tablet as much as possible.

There are menus for Sculpt mesh, Texture Paint, Vertex Paint, Weight Paint, Sculpt Curves, and objects created from the sculpt menu.

How to use

Sculpt mesh

In sculpt mode, menu will open when you press "Shift + U" key (by default, you're able to change from preferences).

Press same key or click empty area to close with do nothing.


The menu is separated from some section as below.

40: Create new primitive object(some paraneters are editable from preferences) / 41: Join Objects / 42: Make backup / 43: Transfer mode / 44: Switch mode

Clicking "16: Grow and Shrink mask" and "17: Smooth and Sharpen mask" will allow you to edit the mask by moving the mouse.

  • Move the mouse to the right to Smooth/Grow, and to the left to Sharpen/Shrink.
  • Confirm with left click, right click, enter key or Esc key.
  • This operation cannot be canceled, so if you wish to do so, please undo after confirming.
  • The amount of mouse movement to change the value can be changed from Preferences.

39: Smart Multires is for when there are too many vertices and getting your Blender slow.

The following operations will be performed automatically:

  • First,system duplicates active object.
  • Remesh the duplicated object.(Be careful that it is remeshed with current voxel size.
    Need to set voxel size to more large than active object's one.)
  • Then, automatically added multiresolution and shrinkwrap modifier.(Default subdivision of multiresolution is 2.)

If you are satisfied, you will apply shrinkwrap modifier.
If not, subdvide more or redo with different voxel size before apply.

It may be fast your Blender.

To set collection for "54: Show or hide collection set by UI", in outliner, right click on collection you want to set and press Set to Show/Hide Collection button.

Color Palette

You can check the active palette from N Panel > Tool > Brush Settings > Color Palette.

Colors are added to the Color History when you perform an action such as painting after changing the Primary or Secondary Color.

If you turn on "Add an existing color to History" in Preferences, colors that have already been added to History will be added in the order they were used.

If brush color usage is Gradient, a list of gradient colors will be displayed at the top of the 3D Viewport.

These are simplifications and do not accurately represent the position on the color ramp.

If you want to change the details, change the color ramp in N Panel > Tool > Brush Settings > Color Picker.


When you drag on the Color Palette, a rectangle will appear over the selected color.

When you release the mouse in this state, the following will happen depending on the brush color usage:


Single Color: Set selected color to secondaty color


Gradient: Open panel for set selected color to secondary or one of gradient

Vertex Paint

In Vertex Paint Mode, menu will open when you press "Shift + U" key (by default, you're able to change from preferences).

Press same key or click empty area to close with do nothing.

Most of them are the same as Texture Paint.

Since gradients cannot be used in Vertex Paint, that part has been removed and the tools have been changed to those used in Vertex Paint.

Note: In Blender 4.3 and above, these are buttons to access each brush rather than tools.

Weight Paint

In Weight Paint Mode, menu will open when you press "Shift + U" key (by default, you're able to change from preferences).

Press same key or click empty area to close with do nothing.

Most of them are the same as Vertex Paint.

Since you cannot use color in Weight Paint, that part is missing and the tool has been changed to that of Weight Paint.

The donut-shaped part that changes the strength has the upper half changing the strength and the lower half changing the weight.

Note: In Blender 4.3 and above, these are buttons to access each brush rather than tools.

Sculpt Curves

In Sculpt Mode for curves, menu will open when you press "Shift + U" key (by default, you're able to change from preferences).

Press same key or click empty area to close with do nothing.

The outermost parts, clockwise from the top right, are "Toggle Select Mode", "Select All", "Deselect All", "Invert Selection", "Grow Selection" and "Toggle Falloff Shape".

Note: In Blender 4.3 and above, these are buttons to access each brush rather than tools.
