Origin to Selected

This addon let you to change object's origin to selected.

日本語のドキュメント

How to use

  • Select object(mesh, curve, surface, armature, or lattice).
  • Go to Edit mode.
  • Select any vertex/edge/face you want to move origin to.
  • Press Shift+Q (default).

If you turn on "Auto Clear Location" in preferences, object will move to world origin by object's new origin.

If you turn on "Auto Go to Object Mode" in preferences, it will automatically go to Object Mode.

You can change shortcut in preferences.
