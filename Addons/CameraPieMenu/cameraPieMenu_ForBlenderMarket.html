Camera Pie Menu

This add-on is a menu that summarizes some camera operations.

日本語のドキュメント

How to use

On the 3D viewport, press Ctrl + W (default)

The available functions differ depending on whether a camera is selected or not.

Functions

  • Camera View
  • Align Active Camera View
  • Lock Camera to View
  • Select Active Camera
  • Select All Camera
  • Select Next Camera
  • Select Previous Camera
  • Set Selected Camera Active
  • Create Camera Aligned to View
  • Show / Hide Passepartout(you can change passepartout alpha value in addon preferences).
  • Frame Camera Bounds
  • Track To / Damped Track
    Add "Track To" or "Damped Track" constraint to active camera.
    Select the target and then the camera, and then press the button to set the target automatically.
  • Composition Guides

Preferences

In Preferences, you can change some settings for this addon.

  • Passepartout Alpha
  • Select Order: Selecting order when you pressing Next or Previous.
  • Select next / previous by case-insensitive:
    If true and Select Order is Name, it select cameras by case-insensitive.
  • Switch to camera view after select next / previous: If true, automatically sets the camera active and goes to camera view when you pressing Next or Previous.
  • Shortcut: You can change shortcut key.
