Bounce Keyframe

This addon converts two keyframes into a bouncy animation.

日本語のドキュメント

How to use

  • Go to Timeline or Graph Editor.
  • Select two keyframes that you want to turn into bounce shape.
    Only one channel should be selected.
  • Press "Bounce" button in header.
  • Adjust parameters.

If the parameter adjustment panel disappears, you can redisplay it by pressing F9 before performing other operations.

Parameters

  • Bounce Count
  • Height: Value of the highest part of the bounce
  • Height Falloff:
    0 = No falloff.
    1 = Top of last bounce will be same as last keyframe value you have selected.
  • Timing Falloff:
    0 = Almost even. As the value increases, they will move backward.
  • Flight Time Ratio: As the value increases, the curve at the top of the bounce becomes smooth.
  • Starts With Rise
  • Snap to Nearest Frame
