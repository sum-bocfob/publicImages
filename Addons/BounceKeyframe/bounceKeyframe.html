Bounce Keyframe

二つのキーフレームを、バウンドする形のアニメーションに変換するアドオンです。

使用方法

  • 二つのキーフレームでアニメーションを作成
  • タイムラインまたはグラフエディターを開く
  • バウンドさせたい二つのキーフレームを選択
    一つのチャンネルのみ選択してください
  • ヘッダーの”Bounce”ボタンをクリック
  • パラメーターを調整

パラメーターを調整するパネルが消えてしまった場合、他の操作をする前ならF9キーで再表示できます。

パラメーター

  • Bounce Count: バウンド回数
  • Height: バウンドの最も高い位置
  • Height Falloff: 高さの減衰
    0 = 減衰なし
    1 = 最後のバウンドの頂点は、選択した最後のキーフレーム値と同じになる
  • Timing Falloff: タイミングの減衰
    0 =ほぼ等間隔
    値を上げると、後方に移動
  • Flight Time Ratio: 値を上げると、バウンドの頂点がスムーズになる
  • Starts With Rise: バウンドが上昇してから開始
  • Snap to Nearest Frame: 近いフレームにスナップ
