Render Multi Objects

レンダリングを繰り返し、複数のオブジェクトに対して、一度で複数の画像/動画を出力します。

複数のオブジェクトをレンダリングしたいとき ”レンダリング→カメラを移動→レンダリング…”を何度も繰り返すのは非常に面倒です。

このアドオンを使用すれば、一度で大量のオブジェクトを個別にレンダリングできます。

※バージョン1.0.1以上は、Blender3.2以降のみ対応

使用方法

  • カメラを配置し、出力パスなどのレンダリング設定をします。(ビューポートレンダリングをする場合でも、カメラの向きが使用されます。)
  • レンダリングしたいオブジェクトを選択します。(アクティブなコレクションを使用する場合は必要ありません。)
  • カメラの位置の基準となるオブジェクトをアクティブにします。(それぞれこのオブジェクトとカメラの位置関係を維持しながらレンダリングされます。)
  • レンダー→Render Multi Objectsをクリックします。
  • ポップアップでオプションを選択し、OKを押してレンダリングを開始します。

レンダリングを途中でキャンセルしたいときには、画像エディター(レンダリングを開始すると立ち上がるウィンドウ)の上のメニューに表示される”Cancel Render”ボタンをクリックしてください。
このボタンは、レンダリングを繰り返す工程を止めますが、現在進行中のレンダリングは止めることができません。
現在進行中のレンダリングを止めたい場合は、Escキーを押してください。

設定項目

  • Object Select Type
    • Selected: 選択しているオブジェクトをレンダリングします。
    • Active Collection: アクティブなコレクション内のすべてのオブジェクトをレンダリングします。 アクティブなコレクションのアイコンは、ハイライトされています。 コレクションをクリックすることで変更できます。
  • Solo Each Object: ほかのオブジェクトを隠し、現在レンダリング中のオブジェクトのみ表示します。(Object Select Typeによってレンダリング予定のオブジェクトにのみ適用されます。)
  • Fit Selection to Camera Frame: カメラの向きはそのままに、オブジェクトがカメラ枠内に収まるようになります。
  • Render Viewport: レンダービューではなく、ビューポートをレンダリングします。
  • Render Animation
  • Fix Camera: 通常カメラは、アクティブオブジェクトとのオフセットを、それぞれのオブジェクトに適用しますが、これをオンにすると、カメラは固定されます。
  • Base Name: レンダリングされたファイルのベースとなる名前。
    • Object Name: オブジェクト名を使用します。
    • Index: 0から始まる番号が割り当てられます。 これは、オブジェクトの選択順やアウトライナーでの順番ではありません。
  • Use Prefix / Suffix: Base Nameの前後に、文字を追加します。
