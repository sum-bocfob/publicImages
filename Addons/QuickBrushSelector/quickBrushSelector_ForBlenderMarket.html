Quick Brush Selector

This addon makes it easy to select brushes for sculpt, Texture Paint and Grease Pencil(Draw Mode) with circular menu.

This makes it easy to load brushes between different blend files and different versions of Blender.

Versions 1.6.1 and above are compatible with Blender 3.4 and above.

日本語のドキュメント
Due to improved functionality of the Sculpt Brush Toolset, the product name has changed to Quick Brush Selector.
Texture Paint mode and Grease Pencil Draw mode functions work with Blender 4.3 and later.
For Blender 4.2 and earlier versions, please use sculpt_brush_toolset_1.2.2.zip, which only works for sculpt brushes.

Features

  • Easily activate brushes with your preferred settings
  • Save multiple toolset(collection of brushes)
  • Customizable UI parameters

How to use

In Sculpt mode, Texture Paint mode or Draw Grease Pencil mode, select QBS in N panel.

Add Toolset / Brush

"Toolset" is collection of brushes.
You can have multiple Toolsets and add them by clicking "Add Toolset".



You can add a brush to the active Toolset or new Toolset by right-clicking on the brush in the AssetShelf of the 3DViewport and selecting "Add to active toolset" or "Add to New Toolset".



You can also add tool like Box Mask, Lasso Hide, Cloth Filter, Move, etc, in Sculpt and Box, Trim, Eyedropper, etc, in Draw(Grease Pencil).

Firstly make the tool you want to add to toolset active, then right-clicking on tool shelf and click "Add Active Tool to Active Toolset / New Toolset".



From Asset Browser, you can add brush to toolset of each mode.
And also, you can add multiple brushes.

Select Toolset / Brush

After adding brush to Toolset, brush selector is shown by pressing Y key(default) on 3DView.
Activate the brush by clicking on the image of the brush you want to select.



When you have multiple toolsets, you can change active toolset by pressing T or U key(default) after open brush selector.
And also chnage by clicking diamond shape above brush selector.



You can close it by pressing the same key you used to open it or by moving mouse far away.

Brushes are arranged clockwise from the top.
You can change the order or delete them from the panel.

Import / Export

In import / export menu, there are 3 ways to import(save) / export(load).

Global Library can be saved / loaded between different files only between the same version of blender.
Only one global library is saved for each blender version.

Note that "Load Global Library" is overwrite current toolset.

The Global Library is saved separately for each mode.

Toolset exports the active toolset to a text file and imports a toolset from a exported text file.

Library imports/exports all tool sets to a text file.

Note that "Import Library" is overwrite current toolset.

Settings

In Settings, there are some settings you can tweak.


If you turn on "Use Simple Panel", brush selector will turn into simple panel with only its name.
Simple panel displays brushes only in active toolset.(You cannot change the toolset outside of the panel.)
The brushes are ordered from top left to bottom.



Icons for the brush selector are generated automatically, but if they are deleted for some reason, you can regenerate them by clicking "Regenerate Icon Images".



In Toolset Settings, you can change Toolset name or order.



There are some settings in Preferences > Add-ons > Quick Brush Selector

"Auto Load Global Library" automatically loads the Global Library in new Blender file.
